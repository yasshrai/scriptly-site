# Install Android Studio

Build amazing Android apps with a clean installation script.

### 1. Clone the repository
```bash
git clone https://github.com/yasshrai/scriptly
```

### 2. Navigate to scripts
```bash
cd scriptly/scripts
```

### 3. Make the script executable
```bash
chmod +x installandroidstudio.sh
```

### 4. Run the installer
```bash
sudo ./installandroidstudio.sh
```
