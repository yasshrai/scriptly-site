# Install IntelliJ IDEA

Expand your development environment with IntelliJ IDEA.

### 1. Clone the repository
```bash
git clone https://github.com/yasshrai/scriptly
```

### 2. Navigate to scripts
```bash
cd scriptly/scripts
```

### 3. Make the script executable
```bash
chmod +x installintellij.sh
```

### 4. Run the installer
```bash
sudo ./installintellij.sh
```
