# Install VS Code

Reliable installations from official sources for Linux x64.

### 1. Clone the repository
```bash
git clone https://github.com/yasshrai/scriptly
```

### 2. Navigate to scripts
```bash
cd scriptly/scripts
```

### 3. Make the script executable
```bash
chmod +x installvscode.sh
```

### 4. Run the installer
```bash
sudo ./installvscode.sh
```
